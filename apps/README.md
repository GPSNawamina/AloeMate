# AloeVeraMate Monorepo

Parent directory for the AloeVeraMate project containing both mobile and server applications.

## Quick Start

See main [README.md](../../README.md) for complete setup instructions.

## Structure

- `mobile/` - React Native Expo mobile application
- `server/` - Python FastAPI backend server

## Scripts

From root directory:

```bash
# Install all dependencies
npm run install:mobile
npm run install:server

# Start mobile app
npm run mobile

# Start backend server  
npm run server
```
